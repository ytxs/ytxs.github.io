# 如何在双击事件中防止单击事件的触发？

描述： 给同一个DOM元素绑定了单击事件又绑定双击事件。

在触发双击事件的时候，会触发以下这些事件，并且先后顺序如下：

双击（dbclick）： mousedown（鼠标按下），mouseup（鼠标移开），click，mousedown，mouseup，click，dblclick 。

在触发单击事件的时候，会触发以下这些事件，并且先后顺序如下：

单击（click）: mousedown，mouseup，click；

如何防止单击事件在双击事件中触发？

可以给单击事件一个延时器，如果两次单击在某个时间之内，就会触发双击函数，不会触发单击函数，如果在那个时间以外，就会触发两次单击而不会触发双击函数

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <script src="../js/jquery.min.js" type="text/javascript"></script>
</head>
<body>
  <div  style="width: 60px;height: 20px;border: 1px solid #cdcdcd;background-color: #eee;border-radius: 3px;cursor: pointer" id="button">button</div>
</body>

<script>
var timeId;
$("#button").off("click")
$("#button").on('click', function () {
  clearTimeout(timeId);
  timeId = setTimeout(function () {
    console.log("单击");
  }, 500);
})
$("#button").off("dblclick")
$("#button").on('dblclick', function () {
  clearTimeout(timeId);
  console.log("双击");
})
</script>
</html>
