    # vue组件间的通信

    # 1. 父组件往子组件传值

    子组件通过props接收父组件传过来的值

    子组件接收到父组件里传过来的值 。如果父组件传过来的值是变化的，那么则在子组件的mounted的生命周期里面可能接收不到。值的变化可以在beforeUpdate和updated生命周期监听的到.

    # 2. 子组件往父组件传值

    子组件通过$emit()函数传递，父组件通过函数方法接收值，也可以通过这个方法来修改父组件中的值。

    eg：

    // 子组件通过this.$emit()传递

if (row._level === 1 && column.property === 'name') {
  this.$emit('show-orgName', row)
  // this.$refs.treeTable.toggleExpanded(row.toggleIndex)
} else if (row._level === 2 && column.property === 'name') {
  this.$emit('show-orgName', row)
} else if (column.property === 'notNum') {
  this.$emit('show-noNum', row.id)
}

    # 3. 非父子组件间的数据传递

    兄弟组件传值eventBus，就是创建一个事件中心，相当于中转站，可以用它来传递事件和接收事件。$emit 传值，$on() 接收值。项目比较小时，用这个比较合适。

    # 5. 父组件修改子组件中的值

    父组件如果要修改子组件中的值，可以给子组件定义一个ref，通过this.$refs['son'].a 去修改

    举个栗子：

    我在父组件中引用了一个userInfo子组件 想修改子组件中的值

    <ShowFromRight ref="panel" @toHidden="hiddenPanel" v-model="togglePanelFlag">
  <UserInfo :editFlag="editFlag" ref="userInfo" :lastLevelOrg="lastLevelOrg" :updateFlag="updateFlag" :settlementModes="settlementModes" :data="clickedUser" @change="(val) => { hasChange = val }" @cancel="hiddenPanel" @update="search"></UserInfo>
</ShowFromRight>
